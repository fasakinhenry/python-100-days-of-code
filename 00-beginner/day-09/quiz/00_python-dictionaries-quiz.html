Question 3

Which line of code will print "Steak"?

order = {
    "starter": {1: "Salad", 2: "Soup"},
    "main": {1: ["Burger", "Fries"], 2: ["Steak"]},
    "dessert": {1: ["Ice Cream"], 2: []},
}

1
2
3
4

Correct Answer

4

print(order["main"][2][0])

Explanation

[2] accesses the value with key 2, [0] gets the first item from the list.
